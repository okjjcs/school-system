جميع الملفات جاهزة ومنظمة بشكل صحيح.

المتطلبات للتشغيل:
  • PHP 7.4 أو أحدث مطلوب
  • SQLite3 مطلوب
  • خادم ويب (Apache/Nginx/PHP Built-in) مطلوب
خطوات التشغيل:
تثبيت PHP
  1. قم بتحميل PHP من php.net
  2. قم بتثبيته وإضافته إلى PATH
  3. تأكد من تفعيل SQLite
تشغيل النظام
  1. افتح Terminal/CMD
  2. انتقل لمجلد المشروع
  3. شغل: php -S localhost:8000
  4. افتح: localhost:8000
بديل سريع:

يمكنك استخدام XAMPP أو WAMP أو MAMP لتشغيل النظام بسهولة:

  • قم بتحميل وتثبيت XAMPP
  • انسخ مجلد المشروع إلى htdocs
  • شغل Apache من لوحة تحكم XAMPP
  • افتح localhost/اسم_المجلد
